# Introdução ao Git e GitHub

## O que é Git?

O Git é um sistema de controle de versão distribuído utilizado para gerenciar alterações em arquivos e projetos de software. Ele permite acompanhar modificações, trabalhar em equipe e recuperar versões anteriores do projeto.

## Principais funções do Git

- Controle de versão de arquivos
- Histórico de alterações
- Trabalho em equipe
- Criação de branches (ramificações)
- Merge de alterações
- Backup do projeto
- Rastreamento de mudanças no código

## Principais comandos do Git

### Configuração inicial


git config --global user.name "Seu Nome"  

git config --global user.email "seuemail@email.com"

## O que é GitHub?

O GitHub é uma plataforma online utilizada para armazenar projetos Git e facilitar o desenvolvimento colaborativo.

- Principais funções do GitHub
- Hospedagem de repositórios
- Compartilhamento de código
- Controle de versões online
- Trabalho em equipe
- Criação de Issues
- Pull Requests
- GitHub Pages
- Backup em nuvem
Integração com ferramentas de desenvolvimento
