# Introdução ao Git e GitHub

## O que é Git?

O Git é um sistema de controle de versão distribuído utilizado para gerenciar alterações em arquivos e projetos de software. Ele permite acompanhar modificações, trabalhar em equipe e recuperar versões anteriores do projeto.

## Principais funções do Git

- Controle de versão de arquivos
- Histórico de alterações
- Trabalho em equipe
- Criação de branches (ramificações)
- Merge de alterações
- Backup do projeto
- Rastreamento de mudanças no código
